Программа принимает на вход параметры вводимые через пробел вида: <путь_к_файлу_источнику> <параметр> <путь_к_файлу_для_вывода>
где
<путь_к_файлу_источнику> - файл содержащий коды фигур и их начальные значения;
<параметр> :
    "-c" - для вывода информации о фигурах в консоль;
    "-f" - параметр для вывода иинформации о фигурах в файл;
<путь_к_файлу_для_вывода> - файл для вывода информации о фигурах.Присутствует только в случае с параметром "-f".
Для корректной работы программы значения фигур должны быть положительными.
